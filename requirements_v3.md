# きょうしん輸送 勤怠連動型給与計算システム 要件定義 v3

## 1. 目的
- MF給与のUI/運用負荷を下げ、管理者が短時間で月次確定できる構成にする。
- 勤怠はHRMOSを継続利用し、給与計算を自社Webで実施する。

## 2. 対象と前提
- 会社: きょうしん輸送株式会社
- 所在地管轄: 北海道 / 所轄税務署: 岩見沢
- 利用者: 管理者1名（まずは単独運用）
- 支払サイクル: 末日締め・翌月20日払い
- 社会保険料徴収: 翌月徴収

## 3. 対象従業員
- 現在の在籍者: 5名以下
- 実装優先: 渡会流雅、渡会羊一、門馬将太
- 退職者含む移行対象: 27名分データ

## 4. 勤務・組織要件
- 所定労働時間: 1日6.7時間
- 月平均所定労働時間: 173.0時間
- 所定労働日数: 月平均26日
- 部門管理: 運送事業 / 作業受託事業
- 職種管理: ドライバー、選果管理、選果作業、労務運行管理、一般事務
- 独自休日: 年始(1/1-1/5)、年末(12/30-12/31)

## 5. 必須機能
- HRMOS API連携（有料プラン前提）
- 月次給与計算（固定給 + 残業 + 控除）
- 所得税自動計算
- 社会保険料自動計算（健保・介護・厚年）
- 雇用保険計算（対象者のみ）
- 有給/休暇管理
- 賞与計算（年2回）
- 給与明細PDF出力
- 部門・職種別集計

## 6. 計算ロジック（MF検証反映）
- 時間単価: （基本給 + 職務手当） / 月平均所定労働時間
- 法定外残業: 時間単価 × 時間 × 1.25
- 法定内残業: 時間単価 × 時間 × 1.00
- 深夜残業: 時間単価 × 時間 × 1.25（別建て加算）
- 丸め: 1円未満切り上げ
- 雇用保険料率: 5.5‰（対象者）
- 介護保険: 40歳以上対象、表示上健保欄へ合算されるケースあり
- 門馬将太: 2025-11以降は役員扱い（雇用保険なし）

## 7. 社保設定値（MF画面反映）
- 健康保険料率: 51.55 / 1,000
- 介護保険料率: 7.95 / 1,000
- 厚生年金保険料率: 91.5 / 1,000
- 子ども・子育て拠出金: 3.6 / 1,000（事業主のみ）

## 8. データ移行
- MF給与CSV（過去給与明細）を基準データ化
- 対象期間: 少なくとも直近12か月
- 比較検証: 新システム計算結果と月次突合

## 9. 画面要件
- ダッシュボード: 総支給、差引支給、残業時間、未確定件数
- 月次計算: 社員別行、内訳展開、残業編集、即時再計算
- 従業員マスタ: 部門/職種/保険区分/役員区分
- 明細出力: PDFプレビュー + 一括出力

## 10. 技術構成（現実解）
- フロント: Next.js
- バックエンド: Node.js
- DB: PostgreSQL（初期PoCはSQLiteでも可）
- 配置: 社内ネットワークのみ

## 11. 開発フェーズ
1. PoC（UI + 計算エンジン + CSV検証）
2. HRMOS連携・従業員マスタ
3. 月次確定・明細PDF
4. データ移行・回帰テスト
5. 本番導入
